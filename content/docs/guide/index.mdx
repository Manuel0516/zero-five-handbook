- **Conventions** — File naming, folder structure, frontmatter rules, and consistency guidelines
- **Writing docs** — The editorial workflow: how to read, write, review, and publish documentation
- **Fumadocs reference** — A complete guide to every component, feature, and API available in this project

Whether you're adding a new page, fixing an existing one, or just learning the ropes, start here.
