## Purpose

This handbook catalogs **everything** installed, configured, and customized on this MacBook. The goal is **reproducibility**: if this machine dies or you buy a new one, you can follow these docs step by step to get back to the exact same state — no guesswork, no forgotten configs.

## Quick start for a new MacBook

If you're setting up a fresh machine, follow the pages in this order:

1. **[Homebrew](/docs/macbook/homebrew)** — Install the package manager first; everything else depends on it
2. **[Terminal](/docs/macbook/terminal)** — Ghostty, Starship, aliases, and fonts
3. **[Git and GitHub](/docs/macbook/git)** — Identity and authentication
4. **[SSH](/docs/macbook/ssh)** — Generate a new key, register it with GitHub and the [VPS](/docs/vps-alpha)
5. **[WireGuard](/docs/macbook/wireguard)** — Install the CLI and join the private VPS network
6. **[Python](/docs/macbook/python)** — Miniforge and the conda environments
7. **[Docker](/docs/macbook/docker)** — Colima container runtime
8. **[LaTeX](/docs/macbook/latex)** — MacTeX distribution
9. **[AI Coding Tools](/docs/macbook/ai-tools)** — Claude Code, Codex, OpenCode
10. **[llama.cpp](/docs/macbook/lama-cpp)** — Local LLM backend
11. **[LocalDrop](/docs/macbook/localdrop)** — Shared clipboard with the Linux desktop
12. **[Applications](/docs/macbook/apps)** — GUI apps and VS Code extensions

> **Tip:** Keep this handbook open on the old MacBook (or on GitHub) while setting up the new one, and check off each section as you complete it.
