## VS Code extensions

Install the full set on a new machine:

```bash
code --install-extension anthropic.claude-code \
     --install-extension openai.chatgpt \
     --install-extension ms-python.python \
     --install-extension ms-python.vscode-pylance \
     --install-extension ms-python.debugpy \
     --install-extension ms-python.vscode-python-envs \
     --install-extension ms-toolsai.jupyter \
     --install-extension marimo-team.vscode-marimo \
     --install-extension ms-vscode.cpptools-extension-pack \
     --install-extension platformio.platformio-ide \
     --install-extension james-yu.latex-workshop \
     --install-extension shd101wyy.markdown-preview-enhanced \
     --install-extension esbenp.prettier-vscode \
     --install-extension ms-azuretools.vscode-containers \
     --install-extension ms-vsliveshare.vsliveshare \
     --install-extension janisdd.vscode-edit-csv \
     --install-extension phplasma.csv-to-table \
     --install-extension meijiahao.vscode-h5-viewer \
     --install-extension ms-ceintl.vscode-language-pack-es \
     --install-extension vscode-icons-team.vscode-icons \
     --install-extension zhuangtongfa.material-theme \
     --install-extension robbowen.synthwave-vscode
```

By area:

| Area | Extensions |
|------|-----------|
| AI | Claude Code, ChatGPT/Codex |
| Python | Python, Pylance, Debugpy, Python Environments, Jupyter, marimo |
| C/C++ / embedded | C/C++ Extension Pack, PlatformIO IDE |
| LaTeX / Markdown | LaTeX Workshop, Markdown Preview Enhanced |
| Data files | Edit CSV, CSV to Table, H5 Viewer |
| Collaboration | Live Share, Containers |
| Formatting | Prettier |
| UI | vscode-icons, One Dark Pro, SynthWave '84, Spanish language pack |
